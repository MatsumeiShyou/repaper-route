# ADR 0008: ハイブリッド・レジリエント統治 (v5.2) の導入

## ステータス
承認済み (Approved) - 2026-03-08

## コンテキスト
ADR 0007 による B-3 条項のハイブリッド化（思考の英語化）に伴い、物理ゲート（特に `epistemic_gate.js`）が日本語マーカーを厳格に要求することで、AI の推論と言語のズレによる不当な反映ブロック（ガバナンス摩擦）が発生するリスクが特定された。

## 意思決定
統治の「堅牢性（Resilience）」と「効率（Efficiency）」を両立させるため、以下の変更を採用する：

1.  **物理ゲートのバイリンガル化**: 
    `epistemic_gate.js` を更新し、日本語マーカーに加えて英語エイリアス（`[Fact]`, `[Self-Criticism]` 等）を合法的な証跡として認める。
    
2.  **構造用タグの言語柔軟化**: 
    憲法および ADR における構造用マーカーについては、日・英どちらの使用も許容する。ただし、コンテンツ自体の日本語品質は引き続き「絶対律」とする。

3.  **トークン節約の努力義務**: 
    `task_boundary` 等のエージェント内部通信においては、リソース保全のため積極的な英語使用を推奨する。

4.  **完了即プロポーズ (Proactive Closure)**:
    封印（Seal）完了後、AI が受動的に待機することを防ぎ、プロジェクトの反映を加速させるため、`/push` の提案を「構造的責務」として定義する。

## 物理的検証 (C-E-V)
- **Negative Proof**: 修正前のゲートが英語マーカーを拒絶することを確認済み。
- **Positive Proof**: 修正後のゲートが日・英・混在の全パターンを Pass することを確認済み。

## 帰結
- 開発摩擦が低減し、AI が自然な推論（英語）を行いつつ、物理的な統治（ゲート）をスムーズに通過可能になる。
- ユーザー向けの最終成果物の日本語品質は維持される。
